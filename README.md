# IELTSClass - 雅思学习系统

## 项目简介

这是一个专为雅思学习者设计的完整学习系统，特别适合：
- **当前水平**: 4.5分
- **目标分数**: 6.5-7.0分（香港/澳门研究生申请要求）
- **学习时间**: 每天1小时 + 额外问题回答时间
- **学习周期**: 预计6-8个月

## 系统特点

✅ **真实题目**: 所有题目均来自官方剑桥雅思真题，确保真实性  
✅ **完整解析**: 每道错题都有完整、清晰的答案和解析  
✅ **问题解答**: 完善的问题回答系统，及时解答学习疑问  
✅ **每日记录**: 每天新建文件夹记录学习日常  
✅ **系统规划**: 完整的学习计划，从4.5分到目标分数  
✅ **时间管理**: 严格按照每天1小时的学习时间设计

## 快速开始

### 1. 创建今日学习文件夹

**方法一：使用脚本（推荐）**

在项目根目录运行：
```bash
# 创建今天的文件夹
bash scripts/create_daily_folder.sh

# 或创建指定日期的文件夹
bash scripts/create_daily_folder.sh 2024-01-15
```

**方法二：手动创建**

1. 在项目根目录创建 `每日学习` 文件夹（如果不存在）
2. 在 `每日学习` 文件夹中创建以日期命名的文件夹，格式：`YYYY-MM-DD`
3. 将 `每日学习模板.md` 复制到该文件夹中，并重命名为 `学习记录.md`

### 2. 开始学习

1. 打开 `每日学习/YYYY-MM-DD/学习记录.md`
2. 按照学习计划完成当天的学习任务
3. 记录学习内容、错题和问题
4. 完成学习后填写学习总结

### 3. 记录问题

遇到任何问题，在 `学习记录.md` 的"今日问题与解答"部分记录，或创建独立的问题文件。

### 4. 记录错题

做错的题目，按照 `错题本系统.md` 的格式详细记录，确保每道错题都有完整解析。

## 学习计划概览

### 第一阶段：基础巩固（第1-2个月）
- **目标**: 从4.5分提升到5.0-5.5分
- **重点**: 熟悉考试格式，建立学习习惯，强化基础词汇和语法

### 第二阶段：能力提升（第3-4个月）
- **目标**: 从5.0-5.5分提升到6.0-6.5分
- **重点**: 听力、阅读技巧强化，写作结构训练，口语话题准备

### 第三阶段：冲刺提高（第5-6个月）
- **目标**: 从6.0-6.5分提升到6.5-7.0分
- **重点**: 高难度题目练习，限时训练，错题深度分析

### 第四阶段：考前冲刺（第7-8个月）
- **目标**: 稳定在6.5-7.0分
- **重点**: 完整模拟考试，错题系统复习，考前心理调整

详细计划请查看 [学习计划.md](学习计划.md)

## 每日学习安排（1小时）

### 时间分配
- **听力**: 15分钟
- **阅读**: 20分钟
- **写作**: 15分钟
- **口语**: 10分钟

### 学习流程
1. **准备阶段**（2分钟）: 创建今日学习文件夹，查看学习计划
2. **学习阶段**（60分钟）: 按照时间分配完成各项练习
3. **记录阶段**（5分钟）: 记录学习内容、错题和问题
4. **问题解答**（额外时间）: 利用额外时间获得问题解答

## 核心功能

### 1. 问题回答系统

遇到任何学习问题，都可以通过问题回答系统获得完整、清晰的解答。

**使用方法**:
- 在每日学习记录中记录问题
- 或在 `问题回答/` 文件夹中创建问题文件
- 按照 `问题回答系统.md` 的格式记录问题

**解答标准**:
- ✅ 直接答案
- ✅ 详细解释
- ✅ 相关例子
- ✅ 易错点提醒
- ✅ 练习建议
- ✅ 相关知识点

详细说明请查看 [问题回答系统.md](问题回答系统.md)

### 2. 错题本系统

系统记录、分析和复习所有做错的题目，确保每道错题都有完整、清晰的解析。

**记录要求**:
- ✅ 题目完整内容
- ✅ 你的答案和正确答案
- ✅ 错误原因分析
- ✅ 正确解题思路
- ✅ 知识点总结
- ✅ 同类题目识别

**复习计划**:
- 每日复习前一天错题
- 每周复习本周错题
- 每月系统复习所有错题
- 考前重点复习高频错题

详细说明请查看 [错题本系统.md](错题本系统.md)

### 3. 题目真实性保证

**所有题目均来自官方真题**:
- Cambridge IELTS 1-18
- Official IELTS Practice Materials
- IELTS Official Guide

**题目标注规范**:
- 明确标注题目来源
- 标注题目编号和类型
- 标注难度等级

**严禁使用**:
- ❌ 非官方模拟题（除非明确标注）
- ❌ 猜测或编造的题目
- ❌ 来源不明的题目

详细说明请查看 [题目真实性说明.md](题目真实性说明.md)


2. **开始学习**
   - 打开 `每日学习/YYYY-MM-DD/学习记录.md`
   - 按照学习计划完成当天的学习任务
   - 记录学习内容和错题

5. **学习总结**
   - 填写今日收获和明日计划

1. **周总结**
   - 回顾本周学习内容

   - 调整下周学习重点

2. **问题回顾**
   - 回顾本周所有问题
   - 确保都已理解

3. **错题复习**
   - 复习本周所有错题
   - 重点复习反复出错的类型

### 每月学习流程

1. **进度评估**
   - 进行完整模拟测试
   - 评估学习进度
   - 调整学习计划

2. **系统复习**
   - 复习本月所有错题
   - 系统回顾所有问题
   - 查漏补缺

## 常见问题

### Q: 如何创建每日学习文件夹？
**A**: 使用脚本 `bash scripts/create_daily_folder.sh` 或手动创建，详细说明见 [创建每日学习文件夹.md](创建每日学习文件夹.md)

### Q: 如何跟踪学习进度？
**A**: 通过每日学习记录、每周总结和每月评估来跟踪进度。建议定期回顾学习记录，评估进步情况。
