# 创建每日学习文件夹 - 使用说明

## 方法一：使用脚本自动创建（推荐）

### Windows系统
在项目根目录下运行：
```bash
# 创建今天的文件夹
bash scripts/create_daily_folder.sh

# 或创建指定日期的文件夹
bash scripts/create_daily_folder.sh 2024-01-15
```

### 手动创建（如果脚本不可用）
1. 在项目根目录下创建 `每日学习` 文件夹（如果不存在）
2. 在 `每日学习` 文件夹中创建以日期命名的文件夹，格式：`YYYY-MM-DD`
   - 例如：`2024-01-15`
3. 将 `每日学习模板.md` 复制到该文件夹中，并重命名为 `学习记录.md`

## 文件夹结构
```
每日学习/
├── 2024-01-15/
│   └── 学习记录.md
├── 2024-01-16/
│   └── 学习记录.md
└── ...
```

## 注意事项
- 文件夹名称必须使用 `YYYY-MM-DD` 格式
- 每天学习前先创建当天的文件夹
- 学习完成后及时填写学习记录
