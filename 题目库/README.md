# 雅思题目库

## 说明
本题目库包含真实的雅思考试题目和完整的答案解析。所有题目均来自官方剑桥雅思真题或官方指南，确保题目的真实性和准确性。

## 题目分类

### 听力题目
- `listening/` - 听力题目和音频文本
- 包含完整答案和详细解析

### 阅读题目
- `reading/` - 阅读文章和题目
- 包含答案、解析和文章翻译

### 写作题目
- `writing/` - 写作Task 1和Task 2题目
- 包含范文和评分标准

### 口语题目
- `speaking/` - 口语Part 1, 2, 3题目
- 包含参考答案和表达技巧

## 使用说明
1. 按照学习计划选择对应难度的题目
2. 完成题目后查看详细解析
3. 记录错题到每日学习记录中
4. 定期复习错题

## 题目来源
- Cambridge IELTS 1-18
- Official IELTS Practice Materials
- IELTS Official Guide
